Community Pantry Connector

Overview

The Community Pantry Connector is a digital platform designed to connect donors, pantry managers, and beneficiaries within a community. The system helps reduce food waste while ensuring that surplus food and essential items reach individuals and families in need.

The platform provides a centralized system for managing donations, tracking inventory, processing beneficiary requests, and coordinating item distribution efficiently.

Objectives
• Facilitate food and essential item donations.
• Reduce food waste through efficient redistribution.
• Improve transparency and accountability in pantry operations.
• Connect donors with beneficiaries through a digital platform.
• Support community efforts toward achieving Zero Hunger.

Features

Donor Module
• Donor registration and authentication.
• Add and manage donated items.
• View donation history.
• Receive notifications on donation status.

Pantry Manager Module
• Manage inventory.
• Approve or reject donations.
• Monitor stock levels.
• Allocate items to beneficiaries.
• Generate reports.

Beneficiary Module
• Register and log into the system.
• Request available items.
• Track request status.
• Receive approval notifications.

Inventory Management
• Real-time stock updates.
• Categorization of items.
• Expiry date tracking.
• Low-stock alerts.

Notification System
• Email/SMS/App notifications.
• Donation confirmations.
• Request approvals and updates.
• Pickup and delivery notifications.

System Architecture
1. Donor Interface
2. Donation Management System
3. Inventory Database
4. Pantry Manager Dashboard
5. Beneficiary Interface
6. Logistics and Distribution System
7. Notification System

Interaction Flow:
Donor → Donation Management → Inventory → Pantry Manager → Beneficiary → Logistics → Notification

Installation
1. Clone the repository.
2. Move the project folder to XAMPP htdocs.
3. Start Apache and MySQL.
4. Import the SQL database in phpMyAdmin.
5. Run using localhost.
